Trying to install the latest Merb from source but having a problem with rake install for merb-more?

(in /usr/local/src/merb/merb-more)

no such file to load -- merb-core/tasks/merb_rake_helper

Assuming you had no problems installing merb-core, then it’s likely that merb-core’s lib directory is only readable by root. Try the install again with sudo, then check the permissions on the gems you just installed to make sure they’re readable to everyone. On OSÂ X that might look something like this:

chmod -R a+rX /Library/Ruby/Gems/1.8/gems/merb*
